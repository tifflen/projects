# projects

## Voice Assistant Project

This repository includes a Python-based voice assistant project in the `voice_assistant/` directory.

Run it from the `voice_assistant` folder with:

```bash
cd voice_assistant
python app.py
```

The subproject implements:
- Flask web app
- Google Cloud Speech-to-Text integration
- Dialogflow webhook endpoint
- SQLite transcript storage
- VOSK offline transcription fallback
